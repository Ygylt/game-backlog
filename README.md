# Game Backlog Tracker

A full-stack web app to track games you want to play, are currently playing, or have completed.

## Features
- Add games with title, platform, status, rating and notes
- Edit and delete games
- Filter by status with color-coded cards
- Persistent data with MongoDB

## Tech Stack
- **Frontend:** React (Vite), deployed on Vercel
- **Backend:** Node.js + Express, deployed on Render
- **Database:** MongoDB Atlas
